Geometry Central and Polyscope

The master branch of polyscope includes geometry-central as a dependency. Often, it's more convenient to keep geometry-central on its own. To do so, you can use branch v02 of polyscope. To add this branch as a submodule, use

git submodule add -b v02 https://github.com/nmwsharp/polyscope.git

If you're working with intrinsic geometry, then the intrinsic branch of geometry-central is probably the one you want. You can add that branch as a submodule using

git submodule add -b intrinsic https://github.com/nmwsharp/geometry-central.git

Upon cloning a repository which uses submodules, you have to initialize the submodules. You do this using

git submodule update --init --recursive
